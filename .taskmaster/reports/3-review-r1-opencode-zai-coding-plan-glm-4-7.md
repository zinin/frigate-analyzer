### Issues

**[IMPORTANT] TelegramNotificationSenderTest.kt:52-66 — Fragile tests using reflection**
Description: Метод `extractReplyMarkup` использует рефлексию для доступа к внутреннему полю `replyMarkup` в обертке `CommonMultipartFileRequest`. Это делает тесты хрупкими — любые изменения в библиотеке tgbotapi или её внутренних API могут сломать тесты без изменения функциональности.
Suggested fix: Рассмотрите возможность использования публичного API tgbotapi для валидации клавиатуры, либо добавьте защитный комментарий о хрупкости теста и необходимости обновления при изменении версии библиотеки.

**[MINOR] TelegramNotificationSender.kt:78-84 — Export button not retried if media group fails**
Description: В ветке `else` (media group) отправка кнопки экспорта происходит без try/catch. Если отправка медиагруппы завершится с ошибкой и прервет выполнение (например, выбросит исключение вместо ретрая), кнопка не будет отправлена. Хотя RetryHelper используется для медиагруппы, порядок выполнения гарантирует, что кнопка будет отправлена только после успешной отправки всех чанков.
Suggested fix: Текущий порядок выполнения корректен благодаря RetryHelper. Для улучшения надежности рассмотрите добавление try-catch вокруг отправки кнопки с логированием ошибок, если требуется гарантировать, что сбой кнопки не критичен.

**[MINOR] TelegramNotificationSender.kt:102-109 — Missing KDoc for new method**
Description: Метод `createExportKeyboard` не имеет документации KDoc, хотя является новым публичным (приватным) API в классе.
Suggested fix: Добавьте KDoc с описанием: `"Creates inline keyboard with export button for video recording"`

**[MINOR] TelegramNotificationSenderTest.kt:108-112 — Missing edge case test for MAX_MEDIA_GROUP_SIZE**
Description: Тест использует только 2 кадра. Не проверяется сценарий с MAX_MEDIA_GROUP_SIZE (10) кадров или больше, где отправляется несколько медиагрупп. В этом случае кнопка отправляется после всех чанков, что корректно, но не протестировано.
Suggested fix: Добавьте тест с 11 кадрами для проверки корректности работы с несколькими медиагруппами.
